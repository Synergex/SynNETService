Name:           SynNETService

Author:         Jerry Fawcett, Senior Developer Support Engineer

Description:    A simple  Synergy .NET Visual Studio project along 
		with a step by step PDF which demonstrates how to
		create a simple Windows service.  

		This service simply writes entries to a file in the
		 Windows (not user's) Temp directory

Requirements:    Visual Studio 2015 and Synergy 10.3.1 (due to Visual Studio
		 2015) or higher. Note support for Service project type added
		 in 9.5.3
